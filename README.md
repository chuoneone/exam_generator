# 🎓 四合一考卷生成

> 一份教材輸入，同步產出學生紙本、語音報讀網頁、教師講解簡報與完整解析。

這不是只會「出題」的 AI 工具，而是一套專為教師設計的教材生成 Skill。教師提供課文、教材、主題或網頁後，AI 會先協助規劃題型與難度，再一次生成課前練習、學生作答、特教報讀與課堂講解所需的完整媒材。

## 🌟 核心亮點：一次出題，四類教材（六個檔案）同步完成

同一份題目與解析會被轉換成四類可直接使用的教材（共六個檔案）：

| 使用情境 | 產出檔案 | 用途 |
| --- | --- | --- |
| 學生紙本作答 | 學生練習卷 PDF | A4 列印，乾淨排版，不含答案 |
| 教師修改與備課 | 學生練習卷 Word | 可在 Word 中調整內容與版面 |
| 特教與視障學生 | 學生點讀有聲 HTML | 句子分段點讀、語速調整、語音選擇與朗讀高亮 |
| 教師課堂投影 | 教師講解簡報 HTML | 逐題投影片、答案控制、TTS、手寫電子白板 |
| 教師查看解析 | 教師解析卷 PDF | 含正確答案與詳細解析，適合列印或備課 |
| 教師編修解析 | 教師解析卷 Word | 可編輯的答案與解析文件 |

換句話說，教師不需要分別製作考卷、報讀網頁和講解簡報；一次設定，就能完成「學生使用」與「教師教學」兩端的媒材。

## 📚 適用情境

- 課堂小考、單元評量與複習卷
- 國文、英文、數學、自然、社會、理化與程式設計
- 國小、國中、高中、大專與成人教育
- 需要放大字體、語音報讀或多元評量的學習者
- 教師希望同時準備紙本、數位學習與課堂講解素材的情境

## 🔄 使用流程

```text
提供課文、教材、主題或網址
          ↓
AI 互動確認學制、題型、題數、配分與字體
          ↓
生成題目、答案、解析與語音朗讀內容
          ↓
同步輸出四類教材（六個檔案）
          ↓
學生作答、學生點讀、教師投影講解
```

## 🛠️ 安裝與使用

### 安裝 Skill

直接在 AI 助理中輸入：

> 幫我安裝這個 Skill：<https://github.com/chuoneone/exam_generator>

安裝完成後，可以直接提出需求，例如：

> 請根據這個單元「高一牛頓第二運動定律」出一份 10 題考卷，包含學生練習卷、教師解析、學生語音報讀網頁和教師講解簡報。

或：

> 請依據《視力與偏見》課文，製作 20 題練習卷，字體 14pt，並產出適合低視能學生使用的報讀版本。

AI 會透過互動問答確認科目、年級、題型、題數、配分、字體大小與解析需求，再在工作區中建立輸出資料夾。

## ♿ 特教與無障礙設計

學生點讀版與教師講解版不是把 PDF 簡單轉成網頁，而是針對實際使用情境設計：

- 句子分段點讀與整題朗讀
- 語速調整與語音角色選擇
- 朗讀中的句子高亮顯示
- 放大字體與適合閱讀的中文字體
- 英文填空底線與數學公式的自然語音處理
- 學生版本嚴格分離答案與解析
- 教師簡報可隱藏或顯示選項、答案與解析
- 內建手寫電子白板，且每一題保留獨立書寫軌跡

## 📐 數學公式與文件品質

- HTML 使用 KaTeX 渲染 LaTeX 數學公式
- 語音朗讀會將公式轉換為自然語句
- Word 使用原生 Office Math（OMML）格式
- PDF 以 A4 版面輸出，隱藏瀏覽器網址、日期與頁首頁尾
- 學生練習卷、學生報讀版不包含答案
- 教師解析卷與教師簡報提供完整解析

## 💡 專案定位

這個專案的核心不是單純增加題目數量，而是把教師最常需要準備的工作整合在一起：

> 出題 → 排版 → 產生紙本 → 製作報讀版本 → 製作教師簡報 → 整理答案解析

讓一份教材可以同時服務不同學習需求，也讓教師把時間放在教學設計，而不是重複整理檔案。
