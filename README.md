# 📝 AI 紙本與有聲數位考卷產生器 (AI Exam Generator Skill)

這是一個專門為國中**數學科**、**國文科**與**英文科**設計的「個別化/特教紙本與有聲數位考卷產生器」的 AI 自訂技能 (Skill)。

當您使用 **Antigravity AI** 程式助理協作時，此技能會被自動載入，引導您透過互動式問答，一鍵生成符合特殊需求與大字排版的考卷與教學媒材。

---

## ✨ 核心特色

### 🗂️ 四合一檔案同步輸出
每次出題皆會於同一個資料夾中同步生成以下四種格式，內容與配分完全一致：

| 檔案 | 說明 |
|------|------|
| `_練習卷.pdf` | 精美 A4 版面，可直接列印發給學生 |
| `_練習卷.docx` | 原生 Word 格式，方便教師後續手動編排 |
| `_教師簡報.html` | 全螢幕投影片，逐題講解專用，內建白板與 TTS |
| `_學生點讀.html` | 學生自主學習用，支援句子分段點讀朗讀 |

### 📐 完美數學公式渲染（KaTeX）
* 所有數學算式（分數、根號、不等式、幾何符號等）以標準 **LaTeX** 格式輸出，透過 **KaTeX** 渲染成印刷等級的精美字體。
* 教師簡報在每次切換題目時皆**重新渲染** DOM 中的數學式，確保分數線（如 $\frac{x-1}{3}$）、希臘字母等符號完美顯示，不顯示 raw LaTeX 原始碼。
* 語音朗讀使用獨立的 `data-read-text` 自然語言屬性（例如「三分之 X 減一」），不受 KaTeX DOM 結構干擾。

### 🎬 教師講解簡報（`_教師簡報.html`）
* 全螢幕投影片版面、黑白主題切換、可顯示/隱藏選項、可顯示/隱藏答案與解析。
* **手寫電子白板**：多色畫筆與橡皮擦，每一題的書寫軌跡獨立記憶，換題不遺失。
* **字體縮放**：工具列 `+`/`-` 按鈕與鍵盤快捷鍵，支援 70% ~ 200% 無段縮放，縮放時手寫軌跡自動重新對齊。
* **整題 TTS 朗讀**：一鍵朗讀整題，或點擊個別句子朗讀，支援語速與語音角色調整。

### 🔊 學生點讀有聲網頁（`_學生點讀.html`）
* **句子分段點讀**：滑鼠移到句子上會顯示藍色虛線，點擊即可朗讀，朗讀時高亮背景。
* **語音控制面板**：語速調節 Slider (0.5x - 2.0x) 與語音角色選擇下拉選單。
* 列印時，點讀按鈕與工具列自動隱藏，不影響紙本美觀。

### 🏫 跨學科客製化排版
* **數學科**：支援「單元範圍出題」，公式以 KaTeX 完美渲染，附帶作答區與詳細解析。
* **國文科**：可指定字型大小（12pt–20pt），支援霞鶩文楷（LXGW WenKai）字體，閱讀測驗支援引文段落。
* **英文科**：預設 Times New Roman 字體，支援單字翻譯、文法選擇、單字挖空與閱讀測驗。

## 🛠️ 如何安裝與使用

### 步驟一：Clone 此倉庫

**工作區安裝（單一專案適用）**：
在您想使用的專案目錄下執行，AI 助理會在開啟此工作區時自動偵測並載入技能：

```bash
git clone https://github.com/chuoneone/exam_generator.git .agents
```

**全域安裝（所有專案通用）**：
若您希望在任何專案中皆能使用此技能，請 Clone 到全域技能目錄：

```bash
# Windows
git clone https://github.com/chuoneone/exam_generator.git "%USERPROFILE%\.gemini\config\skills\exam_generator"

# macOS / Linux
git clone https://github.com/chuoneone/exam_generator.git ~/.gemini/config/skills/exam_generator
```

### 步驟二：告訴 AI 助理您想出什麼考卷

安裝完畢後，只要對 AI 助理說出以下任一指令，就會自動啟動出題流程：

- 「**幫我出一份數學不等式考卷，30 題**」
- 「**幫我出國文愛蓮說的練習卷，20 題**」
- 「**出一份英文文法選擇題，15 題**」

AI 助理會透過互動式問答確認題數、題型、難度、字型大小等設定，最後自動於當前目錄下產出上述四個核心檔案。

> 💡 **注意**：此技能需要搭配 **Antigravity AI 助理**（基於 Google AI Studio Gemini 或 Claude 的代理環境）使用。AI 助理在背景會自動呼叫 Python 腳本、Puppeteer、python-docx 等工具完成檔案生成。
