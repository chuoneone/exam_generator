# 📝 AI 紙本與有聲數位考卷產生器 (AI Exam Generator Skill)

這是一個專門為國中**數學科**、**國文科**與**英文科**設計的「個別化/特教紙本與有聲數位考卷產生器」的 AI 自訂技能 (Skill)。

當您使用 AI 程式助理（如 Gemini 助理）協作時，此技能會被自動載入，引導您透過互動式問答，一鍵生成符合特殊需求與大字排版的考卷。

---

## ✨ 核心特色

### 🖨️ 三合一檔案同步輸出
每次出題皆會於同一個資料夾中同步生成以下三種格式，內容與配分完全一致：
1. **點讀有聲 HTML 網頁 (`.html`)**：內建 Web Speech API 播放引擎，支援學生自主學習。
2. **精美 PDF 考卷 (`.pdf`)**：已最佳化為 A4 無瀏覽器頁首頁尾的乾淨版面，可直接列印。
3. **原生 Word 考卷 (`.docx`)**：方便教師後續手動編排與微調。

### 🔊 數位點讀與語音報讀 (Point-To-Read)
在產出的 HTML 考卷中：
* **句子分段點讀**：滑鼠移到句子上會顯示藍色虛線，點擊即可朗讀該句，且朗讀時會高亮背景。
* **英文空白過濾**：朗讀英文題目時，會自動使用正規表示式過濾掉底線（如 `________`）與括號挖空（如 `(   )`），**直接跳過不唸出 "blank" 或 "underscore"**，確保聽讀流暢。
* **語音控制面板**：網頁頂部提供**語速調節 slider (0.5x - 2.0x)** 與 **語音角色選擇下拉選單**。
* 列印成 PDF 時，點讀按鈕與控制面板會自動隱藏，不影響紙本美觀。

### 🏫 跨學科客製化排版
* **數學科**：支援「單元範圍出題」與貼上題目「同型題改數字模式」，公式以標準 Unicode 呈現，附帶作答區與詳細解析。
* **國文科**：可指定 12pt - 20pt 國字字型大小，支援標楷體，選擇題選項「一個選項一行」排列。
* **英文科**：預設使用 `Times New Roman` 字體，選項「一個選項一行」排列，支援單字翻譯、文法選擇、單字挖空與閱讀測驗。

## 🛠️ 如何安裝與使用

本技能設計為即插即用（Plug-and-Play），只要放在您專案目錄下的 `.agents` 資料夾即可被 AI 助理自動識別：

1. **Clone 本倉庫到您的專案根目錄**：
   在您的專案根目錄下執行：
   ```bash
   git clone https://github.com/chuoneone/exam_generator .agents
   ```

2. **啟動 AI 助理**：
   當您對 AI 說「我想出一份英文考卷」或「幫我出國文雅量考卷」時，AI 助理會自動偵測到此技能，並使用 `ask_question` 互動工具引導您輸入題數、題型、字體大小等設定，最後自動產出檔案。
